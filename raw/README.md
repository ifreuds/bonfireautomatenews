# raw/

Drop unprocessed source material here. Claude **reads but never edits** these files.

Good things to drop in:
- Boss / stakeholder feedback (pasted messages, screenshots of the Lark card)
- Source articles, research PDFs, market reports
- Meeting notes, transcripts, voice-memo dumps
- Draft copy, half-formed ideas
- Data exports

What happens next: Claude **ingests** it — summarizes, files the key ideas into `wiki/` pages, and updates `index.md`.
